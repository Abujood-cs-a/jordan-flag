# Jordan Flag Repository 🇯🇴

Beautifully animated Palestinian flag featuring CSS keyframes and shimmer effects. This project demonstrates how to create realistic fabric movement and lighting using pure web technologies.

## 📝 Project Description
This repository contains a high-quality, animated representation of the flag. It’s designed to be lightweight, responsive, and visually striking, utilizing advanced CSS techniques to achieve a "waving" effect without the need for heavy video files or complex JavaScript.

## ✨ Key Features
- **CSS Keyframes:** Realistic waving and fluttering animation.
- **Shimmer Overlay:** A dynamic light reflection that moves across the flag.
- **Golden Pole:** A 3D-style gradient flagpole.
- **Pure Code:** 100% HTML and CSS (No images or external libraries).

## 🛠️ Technical Breakdown
- **Animations:** Custom `@keyframes` for the waving effect (`flagWave`) and the shimmer (`shimmerMove`).
- **Styling:** Uses `flexbox` for centering and `border-color` tricks to create the iconic triangle.
- **Optimization:** Smooth 60fps performance on most modern browsers.
